# Tsuika

**Tsuika** は、作業に集中している際にふとひらめいたアイデアやメモを素早く記録するためのシンプルなコマンドラインツールです。

メモ帳を開くと他のTo-Doリストや課題に目が行き、集中力が途切れてしまうことがあります。Tsuika はコマンド一つでメモを記録できるため、作業を中断することなく、ひらめきを素早くキャッチできます。

## 特徴

- 🚀 **シンプル**: コマンド一つでメモを記録
- 📝 **タイムスタンプ自動付与**: 各メモに日時が自動で記録されます
- 🔍 **検索機能**: 保存したメモをキーワードで検索
- 📋 **一覧表示**: 保存されているメモを一覧で確認
- 💾 **軽量**: 一時ファイルを使わず、メモリ効率的な実装

## インストール

1. リポジトリをクローンまたはダウンロードします：

```bash
git clone https://github.com/your-username/tsuika.git
cd tsuika
```

2. スクリプトに実行権限を付与します：

```bash
chmod +x tsuika.sh
```

3. スクリプト内の `FILE` 変数を編集して、メモを保存するファイル(.txt)のパスを設定します：

```bash
# tsuika.sh の4行目を編集
FILE="/path/to/your/memo/file.txt"
```

4. （オプション）パスに追加して、どこからでも実行できるようにします：

```bash
# 例: ~/bin にシンボリックリンクを作成
mkdir -p ~/bin
ln -s $(pwd)/tsuika.sh ~/bin/tsuika
export PATH="$HOME/bin:$PATH"
```

## 使用方法

### メモを記録する

引数なしで実行すると、メモ入力モードになります。Enter キーを2回連続で押すと保存されます。

```bash
./tsuika.sh
# または
tsuika
```

実行例：
```
💡 ひらめいたことをメモしてください（Enter 2回で終了）

新しいプロジェクトのアイデア
機能Aと機能Bを組み合わせる
Enter
Enter
✅ 保存しました: /path/to/your/memo/file.txt
```

### メモ一覧を表示する

```bash
./tsuika.sh --list
```

実行例：
```
📋 保存されているメモ一覧:

[1] [2024-01-15 14:30]
新しいプロジェクトのアイデア
機能Aと機能Bを組み合わせる
----------------------------------------

[2] [2024-01-15 15:45]
会議の議事録
重要な決定事項を記録
----------------------------------------
```

### メモを検索する

```bash
./tsuika.sh --search キーワード
```

実行例：
```
./tsuika.sh --search プロジェクト
🔍 「プロジェクト」で検索中...

[1] [2024-01-15 14:30]
新しいプロジェクトのアイデア
機能Aと機能Bを組み合わせる
----------------------------------------
```

### ヘルプを表示する

```bash
./tsuika.sh --help
# または
./tsuika.sh -h
```

## ファイル形式

メモは以下の形式で保存されます：

```
[2024-01-15 14:30]
メモの内容がここに記録されます
複数行のメモも可能です
----------------------------------------
[2024-01-15 15:45]
次のメモエントリ
----------------------------------------
```

## トラブルシューティング

### ファイルが見つからないエラー

```
❌ /path/to/file.txt が見つかりません。
手動でファイルを作成してから、再度実行してください。
```

**解決方法**: スクリプト内の `FILE` 変数で指定したパスにファイルを作成してください。

```bash
touch /path/to/your/memo/file.txt
```

### 実行権限がない

```bash
chmod +x tsuika.sh
```
